# PWA v0.1.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make Counter Weight installable as a PWA on a phone over local WiFi with full offline support.

**Architecture:** Complete the partially-configured `vite-plugin-pwa` setup — add missing manifest fields, generate placeholder PNG icons, and add HTTPS via `@vitejs/plugin-basic-ssl` so the browser's install prompt triggers on LAN. No backend changes; offline works automatically since all data lives in IndexedDB.

**Tech Stack:** Vite 8, vite-plugin-pwa 1.3.0 (Workbox generateSW), @vitejs/plugin-basic-ssl, Node built-ins (zlib) for icon generation.

### Task 2: Generate placeholder icons

**Files:**
- Create: `scripts/gen-icons.mjs`
- Create: `public/icon-192.png` (generated)
- Create: `public/icon-512.png` (generated)

No test needed — output is verified by running the script and checking the files exist.

- [ ] **Step 1: Create the icon generation script**

Create `scripts/gen-icons.mjs` with this content:

```js
import { deflateSync } from 'node:zlib'
import { writeFileSync } from 'node:fs'
import { fileURLToPath } from 'node:url'
import { dirname, join } from 'node:path'

const __dirname = dirname(fileURLToPath(import.meta.url))

function crc32(buf) {
  const table = new Uint32Array(256)
  for (let i = 0; i < 256; i++) {
    let c = i
    for (let j = 0; j < 8; j++) c = c & 1 ? 0xedb88320 ^ (c >>> 1) : c >>> 1
    table[i] = c
  }
  let crc = 0xffffffff
  for (const b of buf) crc = table[(crc ^ b) & 0xff] ^ (crc >>> 8)
  return (crc ^ 0xffffffff) >>> 0
}

function u32(n) {
  const b = Buffer.alloc(4)
  b.writeUInt32BE(n)
  return b
}

function chunk(type, data) {
  const t = Buffer.from(type)
  const len = u32(data.length)
  const crcVal = u32(crc32(Buffer.concat([t, data])))
  return Buffer.concat([len, t, data, crcVal])
}

function solidPNG(size, r, g, b) {
  const sig = Buffer.from([137, 80, 78, 71, 13, 10, 26, 10])
  const ihdr = chunk('IHDR', Buffer.concat([u32(size), u32(size), Buffer.from([8, 2, 0, 0, 0])]))
  // Each row: 1 filter byte (0 = None) + size * 3 RGB bytes
  const row = Buffer.alloc(1 + size * 3)
  for (let x = 0; x < size; x++) {
    row[1 + x * 3] = r
    row[2 + x * 3] = g
    row[3 + x * 3] = b
  }
  const raw = Buffer.concat(Array.from({ length: size }, () => row))
  const idat = chunk('IDAT', deflateSync(raw))
  const iend = chunk('IEND', Buffer.alloc(0))
  return Buffer.concat([sig, ihdr, idat, iend])
}

// #0f172a = rgb(15, 23, 42)
const publicDir = join(__dirname, '..', 'public')
writeFileSync(join(publicDir, 'icon-192.png'), solidPNG(192, 15, 23, 42))
writeFileSync(join(publicDir, 'icon-512.png'), solidPNG(512, 15, 23, 42))
console.log('Generated public/icon-192.png and public/icon-512.png')
```

### Task 3: Complete the PWA manifest and add HTTPS

**Files:**
- Modify: `vite.config.ts`

This task installs `@vitejs/plugin-basic-ssl` and rewrites `vite.config.ts` to complete the manifest and add LAN HTTPS.

- [ ] **Step 1: Install @vitejs/plugin-basic-ssl**

```bash
npm install --save-dev @vitejs/plugin-basic-ssl
```

Expected: package added to `devDependencies` in `package.json`.

- [ ] **Step 2: Replace vite.config.ts**

```ts
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'
import { VitePWA } from 'vite-plugin-pwa'
import basicSsl from '@vitejs/plugin-basic-ssl'

export default defineConfig({
  plugins: [
    react(),
    tailwindcss(),
    basicSsl(),
    VitePWA({
      registerType: 'autoUpdate',
      manifest: {
        name: 'Counter Weight',
        short_name: 'CounterWeight',
        description: 'Local-first countdown timer',
        theme_color: '#0f172a',
        background_color: '#0f172a',
        display: 'standalone',
        orientation: 'portrait',
        start_url: '/',
        icons: [
          { src: '/icon-192.png', sizes: '192x192', type: 'image/png' },
          { src: '/icon-512.png', sizes: '512x512', type: 'image/png' },
        ],
      },
      workbox: {
        globPatterns: ['**/*.{js,css,html,svg,png,ico}'],
      },
    }),
  ],
  server: {
    host: true,
    https: true,
  },
  preview: {
    host: true,
    https: true,
  },
})
```

- [ ] **Step 3: Verify TypeScript compiles**

```bash
npx tsc --noEmit
```

Expected: no output (clean).

- [ ] **Step 4: Commit**


- [ ] **Step 3: Serve over LAN**

```bash

```

Check the terminal output for your LAN URL — it will look like:
```
  ➜  Network: https://192.168.x.x:4173/
```

(Port is 4173 by default; may be 4174/4175 if busy.)

- [ ] **Step 4: Install on phone**

1. Open the `https://192.168.x.x:<port>` URL shown in the terminal on your phone's browser
2. Accept the self-signed cert warning ("Advanced" → "Proceed" on Chrome; "Show Details" → "visit this website" on Safari)
3. Tap the browser menu → "Add to Home Screen" (Chrome/Android) or the share icon → "Add to Home Screen" (Safari/iOS)
4. Open the installed app — it should launch without browser chrome (standalone mode)
5. Turn off WiFi and reopen the installed app — it should load from cache with no network
